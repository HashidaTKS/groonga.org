名前

groonga-query-log

説明

groonga-query-logはGroongaのクエリログを処理するためのライブラリとツール群です。groonga-query-logをライブラリとして使うことでクエリログを処理するプログラムを書くことができます。ツールとして使うことで、使用しているGroongaのクエリを解析したり、クエリログを使ったテストを行うことができます。

インストール

% gem install groonga-query-log
