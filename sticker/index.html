License

The groonga project provides stickers for groonga and groonga related projects such as mroonga. You can use them under the CC-BY 3.0 license. You can also use them under more loose license for distributing printed stickers. See the following paragraph for details. Notation of the author is "The groonga project". You can also use a name of groonga related project for stickers of groonga related projects. Please see also a description for a groonga related project in the below.

If you distribute printed stickers, you don't need to attribute the author. You can print downloaded stickers as is and distribute them without attributing the author. Please print and distribute groonga stickers for your friends!

If you are using groonga on your project, please use our stickers and spread groonga and groonga related projects! :-)
