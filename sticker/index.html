License

The Groonga project provides SVG data of stickers for Groonga and Groonga related projects such as Mroonga. You can use them under the CC-BY 3.0 license. You can also use them under more loose license for distributing printed stickers. See the following paragraph for details. Notation of the author is "The Groonga Project". You can also use a name of Groonga related project for stickers of Groonga related projects. Please see also a description for a Groonga related project in the below.

If you distribute printed stickers, you don't need to attribute the author. You can print downloaded stickers as is and distribute them without attributing the author. Please print and distribute Groonga stickers for your friends!

If you are using Groonga on your project, please use our stickers and spread Groonga and Groonga related projects! :-)
