8.11.8. rand¶

8.11.8.1. 名前¶

rand - 乱数を生成する

8.11.8.2. 書式¶

rand([max])

8.11.8.3. 説明¶

groonga組込関数の一つであるrandについて説明します。組込関数は、script形式のgrn_expr中で呼び出すことができます。

rand() 関数は 0 から max の間の疑似乱数整数を返します。

8.11.8.4. 引数¶

max

返値の最大値を指定します。省略した場合は RAND_MAX が指定されたものとみなされます。

8.11.8.5. 返値¶

0 と max の間の数を表すInt32型の値を返します。

8.11.8.6. 例¶

rand(10)
3
