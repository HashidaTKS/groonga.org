7.1.7. groonga-suggest-create-dataset¶

7.1.7.1. NAME¶

groonga-suggest-create-dataset - Defines schema for a suggestion dataset

7.1.7.2. SYNOPSTIS¶

groonga-suggest-create-dataset [options] DATABASE DATASET

7.1.7.3. DESCTIPION¶

groonga-suggest-create-dataset creates a dataset for Suggest. A database has many datasets. This command just defines schema for a suggestion dataset.

This command generates some tables and columns for Suggest.

Here is the list of such tables. If you specify ‘query’ as dataset name, following ‘_DATASET’ suffix are replaced. Thus, ‘item_query’, ‘pair_query’, ‘sequence_query’, ‘event_query’ tables are generated.

  • event_type

  • bigram

  • kana

  • item_DATASET

  • pair_DATASET

  • sequence_DATASET

  • event_DATASET

  • configuration
