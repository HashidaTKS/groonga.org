7.8.3. TokenBigramIgnoreBlank¶

7.8.3.1. Summary¶

TokenBigramIgnoreBlank is similar to TokenBigram. The difference between them is blank handling. TokenBigramIgnoreBlank ignores white-spaces in continuous symbols and non-ASCII characters.

7.8.3.2. Syntax¶

TokenBigramIgnoreBlank hasn’t parameter:

TokenBigramIgnoreBlank

7.8.3.3. Usage¶

You can find difference of them by 日 本 語 ! ! ! text because it has symbols and non-ASCII characters.
