12.3.2. How to build Groonga at the repository¶

This document describes how to build Groonga at the repository for each build system. You can choose GNU Autotools or CMake if you develop Groonga on GNU/Linux or Unix (*BSD, Solaris, OS X and so on). You need to use CMake if you develop on Windows.
