2.2. Mac OS X¶

このセクションではMac OS X上でGroongaをインストールする方法を説明します。 MacPorts か Homebrew を使ってインストールできます。

2.2.1. MacPorts¶

インストール:

% sudo port install groonga

2.2.2. Homebrew¶

インストール:

% brew install groonga

MeCab をトークナイザーとして使いたいときは、 --with-mecab オプションを指定してください。

% brew install groonga --with-mecab

それからMeCabの辞書をインストール・設定します。

インストール:

% brew install mecab-ipadic

設定:

% sed -i '' -e 's,dicrc.*=.*,dicrc = /usr/local/lib/mecab/dic/ipadic,g' /usr/local/etc/mecabrc

2.2.3. ソースからビルド¶

Xcode をインストールしてください。

ソースをダウンロードします:

% curl -O https://packages.groonga.org/source/groonga/groonga-10.0.5.tar.gz
% tar xvzf groonga-10.0.5.tar.gz
% cd groonga-10.0.5

configureを実行します（ configure のオプションについては configure を参照してください）:

% ./configure

ビルド:

% make -j$(/usr/sbin/sysctl -n hw.ncpu)

インストール:

% sudo make install
