7.9.2.1. TokenFilterNFKC100¶

7.9.2.1.1. 概要¶

このトークンフィルターは、 NormalizerNFKC100 と同じオプションでカタカナのトークンをひらがなに変換できます。このトークンフィルターは、トークンの読みがなをひらがなとして取得したい時に便利です。

7.9.2.1.2. 構文¶

TokenFilterNFKC100 は引数が一つあります。

TokenFilterNFKC100("unify_kana", true)

7.9.2.1.3. 使い方¶

7.9.2.1.4. 簡単な使い方¶

以下は、 TokenFilterNFKC100 の例です。 リンゴ を りんご へ変換しています。

実行例:

tokenize TokenDelimit "リンゴ"   --token_filters 'TokenFilterNFKC100("unify_kana", true)'
# [
#   [
#     0,
#     1545901643.191951,
#     0.0003898143768310547
#   ],
#   [
#     {
#       "value": "りんご",
#       "position": 0,
#       "force_prefix": false,
#       "force_prefix_search": false
#     }
#   ]
# ]

以下のように、 TokenFilterNFKC100 はひらがなと漢字のトークンは変換しません。

実行例:

tokenize TokenDelimit "りんご 林檎"   --token_filters 'TokenFilterNFKC100("unify_kana", true)'
# [
#   [
#     0,
#     1545901725.412921,
#     0.0003237724304199219
#   ],
#   [
#     {
#       "value": "りんご",
#       "position": 0,
#       "force_prefix": false,
#       "force_prefix_search": false
#     },
#     {
#       "value": "林檎",
#       "position": 1,
#       "force_prefix": false,
#       "force_prefix_search": false
#     }
#   ]
# ]

7.9.2.1.5. 高度な使い方¶

TokenFilterNFKC100 と TokenMecab の use_reading オプションを組み合わせることで、入力文字列を全てひらがなとして出力できます。

実行例:

tokenize   'TokenMecab("use_reading", true)'   "私は林檎を食べます。"   --token_filters 'TokenFilterNFKC100("unify_kana", true)'

#     0,

#     0.0003833770751953125
#   ],

#       "force_prefix_search": false
#     },
#     {
#       "value": "は",
#       "position": 1,
#       "force_prefix": false,
#       "force_prefix_search": false
#     },
#     {
#       "value": "りんご",
#       "position": 2,
#       "force_prefix": false,
#       "force_prefix_search": false
#     },
#     {
#       "value": "を",
#       "position": 3,

7.9.2.1.6. 引数¶

7.9.2.1.6.1. 必須引数¶

必須の引数 unify_kana があります。

7.9.2.1.6.1.1. unify_kana¶

カタカナのトークンをひらがなに変換します。
