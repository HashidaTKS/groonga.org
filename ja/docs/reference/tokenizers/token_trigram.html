7.8.17. TokenTrigram¶

7.8.17.1. 概要¶

TokenTrigram は TokenBigram と似ています。違いはトークンの単位です。

7.8.17.2. 構文¶

TokenTrigram には、引数がありません。

TokenTrigram

7.8.17.3. 使い方¶

ノーマライザーを使っている場合は TokenTrigram はASCIIの文字には空白区切りのようなトークナイズ方法を使います。非ASCII文字にはトリグラムのトークナイズ方法を使います。

TokenTrigram が非ASCII文字をトークナイズすると、以下の例のように TokenTrigram は各トークンが3文字となります。

実行例:

tokenize TokenTrigram "日本語の勉強" NormalizerAuto
# [
#   [
#     0,
#     1546586185.123834,
#     0.0003123283386230469
#   ],
#   [
#     {
#       "value": "日本語",
#       "position": 0,
#       "force_prefix": false,
#       "force_prefix_search": false
#     },
#     {
#       "value": "本語の",
#       "position": 1,
#       "force_prefix": false,
#       "force_prefix_search": false
#     },
#     {
#       "value": "語の勉",
#       "position": 2,
#       "force_prefix": false,
#       "force_prefix_search": false
#     },
#     {
#       "value": "の勉強",
#       "position": 3,
#       "force_prefix": false,
#       "force_prefix_search": false
#     },
#     {
#       "value": "勉強",
#       "position": 4,
#       "force_prefix": false,
#       "force_prefix_search": false
#     },
#     {
#       "value": "強",
#       "position": 5,
#       "force_prefix": false,
#       "force_prefix_search": false
#     }
#   ]
# ]
