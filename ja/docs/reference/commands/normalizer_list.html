7.3.39. normalizer_list¶

7.3.39.1. 概要¶

normalizer_list コマンドはデータベースに登録されているノーマライザーの一覧を返します。

7.3.39.2. 構文¶

このコマンドに引数はありません:

normalizer_list

7.3.39.3. 使い方¶

以下は簡単な使用例です。

実行例:

normalizer_list
# [
#   [
#     0,
#     1337566253.89858,
#     0.000355720520019531
#   ],
#   [
#     {
#       "name": "NormalizerAuto"
#     },
#     {
#       "name": "NormalizerNFKC51"
#     }
#   ]
# ]

データベースに登録されているノーマライザーの一覧を返します。

7.3.39.4. 戻り値¶

normalizer_list コマンドはノーマライザーの一覧を返します。各ノーマライザーは属性を持っています。例えば名前です。将来、属性は増えるかもしれません。:

[HEADER, normalizers]

HEADER

HEADER については 出力形式 を参照してください。

normalizers

normalizers はノーマライザーの配列です。ノーマライザーは次の属性を持つオブジェクトです。

名前

説明

name

ノーマライザー名。
