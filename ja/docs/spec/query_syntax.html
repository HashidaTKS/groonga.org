8.1. クエリ構文¶

TODO: revised.

一般的なWebページの検索フォームで使われるような書式を使って検索条件を指定することができます。これをクエリ構文と呼びます。

クエリ構文は「条件式」と条件式を組み合わせる「結合式」からなります。

8.1.1. 条件式¶

以下の条件式が使用できます。

8.1.1.1. 文字列¶

全文検索条件(デフォルト検索対象カラムの値が指定された文字列を含んでいる)

8.1.1.2. "文字列"¶

フレーズ検索条件(デフォルト検索対象カラムの値が指定されたフレーズを含んでいる)

8.1.1.3. カラム名:値¶

一致条件( カラム値 == 値 )

8.1.1.4. カラム名:!値¶

不一致条件( カラム値 != 値 )

8.1.1.5. カラム名:<値¶

比較条件( カラム値 < 値 )

8.1.1.6. カラム名:>値¶

比較条件( カラム値 > 値 )

8.1.1.7. カラム名:<=値¶

比較条件( カラム値 <= 値 )

8.1.1.8. カラム名:>=値¶

比較条件( カラム値 >= 値 )

8.1.1.9. カラム名:@文字列¶

全文検索条件(カラム値が指定された文字列を含んでいる)

8.1.2. 結合演算子¶

複数の条件式を結合するために以下の演算子が使用できます。

8.1.2.1. a OR b¶

論理和( a と b といずれかの条件がマッチする)

8.1.2.2. a + b¶

論理積( a と b の両方がマッチする)

8.1.2.3. a - b¶

a にマッチし、 b にはマッチしない

8.1.2.4. (...)¶

複数の条件をまとめる
