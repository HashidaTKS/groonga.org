5.2. HTTP¶

Groongaは2つのHTTPサーバー実装を提供しています。

  • groonga

  • groonga-httpd

groonga はシンプルな実装です。高速に動作しますがHTTPの機能の多くは実装されていません。コマンドラインオプションをいくつか指定するだけで動くのでGroongaを簡単に試すことができます。

groonga-httpd は nginx をベースにした実装です。この実装も高速に動作します。しかも多くのHTTPの機能を使えます。
