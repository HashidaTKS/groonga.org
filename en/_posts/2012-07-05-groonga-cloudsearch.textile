h2. Groonga CloudSearch 1.0.0 has been released

The initial version of the "Groonga CloudSearch":http://gcs.groonga.org/ has been released!

Groonga CloudSearch is a full text search system based on groonga, including "Amazon CloudSearch":http://aws.amazon.com/en/cloudsearch/ compatible APIs. It will help your development if you are planning to build new Web apps with the Amazon CloudSearch.
